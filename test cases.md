**User story \-** As a user, I want to have an option to share my Instagram stories into my Facebook account, so that I can manage what is posted

**Test cases** 

| ID | Test Case Title | Preconditions | Steps | Expected Result |  |
| ----- | ----- | ----- | ----- | ----- | ----- |
| TC01 | Check that a “Settings and activity” screen can be opened | User is logged into Instagram User is on the “Main” screen  | Tap the hamburger icon "☰" on the "Main" screen | The "Settings and activity" screen is opened |  |
| TC02 | Check that a “Crossposting” screen can be opened  | User is logged into Instagram User is on the “Settings and activity” screen  | Tap the "Crossposting" option | The "Crossposting" screen is opened |  |
| TC03 | Check that a “Sharing across profiles” popup menu can be opened  | User is logged into Instagram and Facebook User’s Instagram and Facebook accounts are connected User is on the “Crossposting” screen  | Tap the "Accounts Center" option | The "Sharing across profiles" popup menu is opened |  |
| TC04 | Check a “Your Instagram story” toggle can be switched ON  | User is logged into Instagram and Facebook User’s Instagram and Facebook accounts are connected User is on the “Sharing across profiles” popup menu A “Your Instagram story” toggle is switched off by default  | Switch on a "Your Instagram story" toggle | A "Your Instagram story" toggle is in the ON position |  |
| TC05 | Verify that an Instagram story is shared on Facebook automatically  | User is logged into Instagram and Facebook User’s Instagram and Facebook accounts are connected A “Your Instagram story” toggle is switched on | Add a new story on Instagram | The story is shared on Facebook automatically |  |
